= Aktualisieren Sie die vCenter Server-Anmeldeinformationen in ONTAP tools
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Sie können die Anmeldeinformationen der vCenter Server-Instanz über die Wartungskonsole aktualisieren.

.Bevor Sie beginnen
Sie müssen über Anmeldedaten für Wartungsbenutzer verfügen.

.Über diese Aufgabe
Wenn Sie die Anmeldeinformationen für den vCenter Server nach der Bereitstellung geändert haben, aktualisieren Sie diese mit diesem Verfahren.

.Schritte
. Öffnen Sie über den vCenter Server eine Konsole für VASA Provider.
. Melden Sie sich als Wartungbenutzer an.
. Geben Sie ein `2` , um das Menü Systemkonfiguration auszuwählen.
. Eingeben `8` um die vCenter-Anmeldeinformationen zu ändern.
