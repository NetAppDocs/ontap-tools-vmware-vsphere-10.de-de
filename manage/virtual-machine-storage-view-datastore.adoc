= Ansicht des Speichers virtueller Maschinen in ONTAP tools
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Die Speicheransicht zeigt die Liste der vVols an, die die virtuelle Maschine erstellt.


NOTE: Diese Ansicht gilt für VMs mit mindestens einer Festplatte aus einem von ONTAP tools for VMware vSphere verwalteten vVols Datenspeicher.

.Schritte
. Gehen Sie vom vSphere-Client zur virtuellen Maschine.
. Wählen Sie im rechten Fensterbereich die Registerkarte *Monitor* aus.
. Wählen Sie *NetApp ONTAP Tools* > *Speicher*. Die *Speicher*-Details werden im rechten Fensterbereich angezeigt. Sie können die Liste der VVols anzeigen, die auf der VM vorhanden sind.
+
Sie können die Option „Spalten verwalten“ verwenden, um verschiedene Spalten ein- oder auszublenden.
