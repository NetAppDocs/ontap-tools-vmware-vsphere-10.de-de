= Benutzerkennwort der Anwendung zurücksetzen
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Führen Sie diese Schritte aus, um das für die SRA- und VASA-Provider-Registrierung bei vCenter Server erforderliche Anwendungsbenutzerkennwort mithilfe der ONTAP tools for VMware vSphere zurückzusetzen.

.Schritte
. Öffnen Sie einen Webbrowser und navigieren Sie zu: `\https://<ONTAPtoolsIP>:8443/virtualization/ui/`
. Melden Sie sich mit den Administratoranmeldeinformationen an, die während der Bereitstellung der ONTAP -Tools konfiguriert wurden.
. Wählen Sie in der Seitenleiste *Einstellungen* aus.
. Auf der Seite *VASA/SRA-Zugangsdaten* wählen Sie *Passwort zurücksetzen*.
. Geben Sie das neue Passwort ein und bestätigen Sie es.
. Wählen Sie *Zurücksetzen*, um das neue Passwort anzuwenden.
